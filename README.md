# Vítor One - Quartz v4

Site pessoal integrado ao Obsidian.

## Opções de Design (Log de Referência)

Estas opções foram discutidas para evolução do UX do site:

1. **Opção 1: O Portfólio Acadêmico (Atual)**
   - Navegação principal no topo (Header).
   - Sidebar esquerda focada apenas no Perfil (Bio/Social).
   - Sem Explorer redundante na lateral.
   - Ideal para um site limpo e profissional.

2. **Opção 2: O Jardim Digital**
   - Navegação concentrada na Sidebar esquerda.
   - Topo limpo apenas com Busca e Modo Escuro.
   - Ideal para quem prioriza a exploração de notas interconectadas.

3. **Opção 3: O Híbrido Funcional**
   - Topo para páginas estáticas (CV, Publicações).
   - Sidebar para exploração profunda de notas (Explorer).
   - Ideal para grandes volumes de conteúdo misto.
